# ratemyidea.ai

AI-powered business idea validator by AI Norte.

## Current state

- Stack: Next.js App Router, TypeScript, Supabase, Stripe, Anthropic, Serper.
- Production domain: `https://ratemyidea.ai`
- Product mode: controlled sale / real-user feedback, not broad public launch.
- Monetization:
  - Free: limited daily evaluations.
  - Pro: `$9/mo` for unlimited evaluations, saved history, benchmark, and AI next-step plans.
  - Market Study: `$49` one-time checkout with generated report and PDF download.

## Important product contracts

- The basic evaluation should return a full useful result; do not lock core fields.
- Language follows the prompt language where possible. If the UI is English but the idea is Spanish, the analysis should be Spanish.
- Market Study preview must not spend on deep research before purchase. The unlocked preview may only reuse data already generated by the basic analysis: summary, score, category cards, strengths/risks, and next-step style signals.
- Market Study deep sections remain visible but locked/blurred until checkout: Market Analysis, Competitor Analysis, Target Audience, Go-to-Market, Financial Projections.
- Final Market Study delivery is the `/study` report with `Download PDF` / `window.print()` as the primary delivery behavior.
- Product tone for Market Study: hybrid C — light consulting report with AI Norte branding, not a generic web-card mockup and not a hype-heavy ebook.

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Verification gates

Run these before committing meaningful changes:

```bash
npm run test:smoke
npm run lint
npm run build
```

The smoke script is intentionally source-level and guards key product contracts around pricing, auth, language, share-credit, Pro history, and Market Study delivery.
